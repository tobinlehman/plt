== Description ==

Provides the ability to disbale and re-enable WordPress Canonical URL Redirection by enabling and disabling this plugin. When the plugin is enbaled, 
WordPress Canonical URL Redirection is disabled, even when it is turned on in the WordPress site settings.

== Installation ==

Option 1: Download from WordPress.org through WordPress Plugin Installer

1. Go to your plugins page and click the "Add New" button.
2. Search for the "Disable Canonical URL Redirects" plugin.
3. Once found, click on the link from the search page to go to the installation page.
4. Click the "Install Now" button to install the plugin from the WordPress site.
5. Activate the Plugin to Disable Canonical URL Redirects.


Option 2: Manually download the Plugin and upload it to your WordPress site: <br/>
1. Download the Plugins .ZIP file. <br/>
2. Extract the .ZIP file so that the files are in the root folder of the expanded folder. <br/>
3. Upload the expanded plugin folder to your site at: /wp-content/plugins/ folder. <br/>
4. Activate the Plugin to Disable Canonical URL Redirects. <br/>
